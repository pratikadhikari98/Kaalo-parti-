# कालिका मानवज्ञान माध्यमिक विद्यालय — Website Structure

## 📁 फाइल संरचना

```
kalika-school/
├── index.html              ← मुख्य वेबसाइट (यो नछोनुहोस्)
├── photos/
│   └── blackboard.png      ← कालोपाटीको फोटो
└── news/
    ├── news1.json          ← सूचना कार्ड १
    ├── news2.json          ← सूचना कार्ड २
    ├── news3.json          ← सूचना कार्ड ३
    ├── news4.json          ← सूचना कार्ड ४
    └── news10.json         ← कालोपाटीमा देखिने विशेष सूचना ⭐
```

---

## ✏️ दैनिक सूचना कसरी फेर्ने?

### 🖼️ कालोपाटीमा देखिने सूचना (news10.json)
```json
{
  "title": "विशेष सूचना",
  "date": "२०८३/०३/१२",
  "lines": [
    "पहिलो लाइन यहाँ लेख्नुहोस्।",
    "दोस्रो लाइन यहाँ।",
    "तेस्रो लाइन।",
    "चौथो लाइन।"
  ],
  "body": "सम्पूर्ण सूचना एकै ठाउँमा।",
  "footer": "विद्यालय प्रशासन"
}
```
- **`date`** — मिति बदल्नुहोस् (नेपाली अंकमा)
- **`lines`** — कालोपाटीमा देखिने लाइनहरू (४ वटासम्म राख्नुहोस्)
- **`title`** — शीर्षक

### 📋 कार्ड सूचनाहरू (news1.json, news2.json ...)
```json
{
  "badge": "📢 सूचना",
  "badgeColor": "#dbeafe",
  "badgeText": "#1e3a5f",
  "date": "असार १५, २०८२",
  "title": "सूचनाको शीर्षक",
  "body": "सूचनाको विवरण यहाँ लेख्नुहोस्।",
  "footer": "विद्यालय प्रशासन"
}
```

## ➕ नयाँ सूचना कार्ड थप्ने (news5, news6 ...)

1. `news/news5.json` बनाउनुहोस् (माथिको format मा)
2. `index.html` खोल्नुहोस्, यो लाइन खोज्नुहोस्:
   ```js
   const FILES = [
     'news/news1.json',
     ...
   ];
   ```
3. `'news/news5.json',` थप्नुहोस्
